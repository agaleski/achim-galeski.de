Web-Entwicklung

Mit Composer als Package-manager eine Runde Sache.

PHP ist meine Hassliebe. Mein KnowHow kommt hier am stärksten zur Geltung. Mit Composer als Packagemanager eine Runde Sache. Mit PHP 7 zeigt sich, PHP war nie tot und lebt gesünder denn je zuvor.

JavaScript ist aus unserer interaktiven, mobilen Welt nicht mehr wegzudenken. Mit AngularJS nun endlich erwachsen geworden und dank jQuery seit Jahren universell zugänglich.

Mit HTML verbindet mich eine lange Leidens- und Arbeitszeit. HTML 5 ist der zwingende Schritt um den Markup Code für die Zukunft zu rüsten. "Keep it Simple and Short" lautet die aktuelle Devise!

Css ist nicht nur das Design-All-In-One-Wonder, es bietet heute dank modernsten techniken in CSS3 die möglichkeit, viel JavaScript zuhause zu lassen, wenn man ansprechende UI Interaktionen schaffen will.
